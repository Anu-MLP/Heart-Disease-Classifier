# Heart Disease Classifier ❤️

## Project Overview
This project uses Machine Learning to predict whether a person is likely to have heart disease based on medical information.

## Dataset
Dataset contains 1025 patient records with 13 features and 1 target variable.

Features include:
- Age
- Sex
- Chest pain type
- Blood pressure
- Cholesterol
- Heart rate
- ECG results
- Other medical measurements

## Machine Learning Models Used

### Logistic Regression
Accuracy: 79.5%

### K-Nearest Neighbors (KNN)
Accuracy: 73.2%

## Model Evaluation
Used:
- Confusion Matrix
- Accuracy Score
- Classification Report

## Prediction Example
The model can predict:
- 0 → No heart disease
- 1 → Heart disease
